# website-react

Personal site built with **React + Vite**.  
Showcasing subtle animations, smooth text effects, and reusable UI components.

![Preview Screenshot](https://raw.githubusercontent.com/sudo-Penguin-Lalala/website-react/refs/heads/main/public/preview.png)

---

## ✦ Features
- Animated backgrounds (Squares / Glitch / CurvedLoop)
- Dynamic text reveal using `DecryptedText`
- Responsive layout optimized for desktop and mobile
- Built with React + Vite for a fast development experience
