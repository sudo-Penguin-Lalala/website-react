# website-react

Personal site built with **React + Vite**.

![Preview Screenshot](https://raw.githubusercontent.com/sudo-Penguin-Lalala/website-react/refs/heads/main/public/preview.png)

---

## Features

**Animations & Effects**
- `AnimatedGrid` — Animated grid background with gradient lines
- `ShapeGrid` — Pastel geometric shape patterns
- `ShinyText` — Shimmering text effect on hover
- `LogoLoop` — Infinite scrolling logo animation
- `GradualBlur` — Progressive blur reveal effect
- `ClickSpark` — Spark particles on click interactions

**Components**
- `Button` — Reusable button with border glow effect
- `CircularText` — Circular text layout for footer/accents
- `Icon` — Custom icon component

**Stack**
- React 19 + Vite
- Motion (Framer Motion alternative)
- GSAP for advanced animations
- OGL for WebGL effects
- Parallax tilt effects

---

